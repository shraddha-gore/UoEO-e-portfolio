Exercise 15.1:

Write a definition for a class named Circle with attributes center and radius, where center is a Point object and radius is a number.
Instantiate a Circle object that represents a circle with its center at (150, 100) and radius 75.
Write a function named point_in_circle that takes a Circle and a Point and returns True if the Point lies in or on the boundary of the circle.

Code: 
              
import math


class Point:
	def __init__(self, x, y):
		self.x = x
		self.y = y


class Circle:
	def __init__(self, center, radius):
		self.center = center
		self.radius = radius


def point_in_circle(circle, point):
	dist = math.sqrt((point.x - circle.center.x) ** 2 + (point.y - circle.center.y) ** 2)
	return dist <= circle.radius


# Create objects
circle = Circle(Point(150, 100), 75)
point1 = Point(160, 100)
point2 = Point(210, 180)

# Check if the points are in the circle
print(point_in_circle(circle, point1))  # Expected output: True
print(point_in_circle(circle, point2))  # Expected output: False
Exercise 15.2:

Write a function called draw_circle that takes a Turtle and a Circle and draws the Circle.

Code: 
              
import turtle


class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y


class Circle:
    def __init__(self, center, radius):
        self.center = center
        self.radius = radius


def draw_circle(t, circle):
    t.circle(circle.radius)


# Draw the circle
t = turtle.Turtle()
circle = Circle(Point(100, 75), 50)
draw_circle(t, circle)
turtle.done()
